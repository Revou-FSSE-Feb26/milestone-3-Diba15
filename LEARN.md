# Learn Next JS

## Description

File MD ini dibuat bertujuan untuk belajar mengenai Next JS.

## Apa yang perlu dilakukan ketika baru membuat project Next JS

- Install Next JS
- Setup Next JS nya dengan
    ```bash
    Would you like to use TypeScript? Yes
    Which linter would you like to use? ESLint / Biome
    Would you like to use React Compiler? No
    Would you like to use Tailwind CSS? Yes
    Would you like your code inside a `src/` directory? Yes  Agar kode lebih rapi dan mudah dibaca
    Would you like to use App Router? (recommended) Yes Untuk kemudahan routing
    Would you like to customize the import alias (`@/*` by default)? No
    Would you like to include AGENTS.md to guide coding agents to write up-to-date Next.js code? No Selama kita tidak memerlukan AI dalam membantu
    ```
- Buat folder-folder penting yang akan digunakan dalam project Next JS (components, assets, utils, hooks, lib, types)
- Buat folder context jika ingin membuat aplikasi menggunakan global state
- Buat folder api jika ingin membuat aplikasi menggunakan API
- Buat folder data jika ingin membuat aplikasi menggunakan dummy data
- Hapus styling bawaan next js dan ganti styling yang ingin kita gunakan
